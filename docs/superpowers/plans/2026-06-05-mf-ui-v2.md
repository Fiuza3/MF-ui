# MF-UI v2.0.0 — Plano de Implementação

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reescrever `@mf/ui` como design system de nível tech-lead: tokens W3C completos, acessibilidade WCAG 2.2 AA, API normalizada em inglês e bump de versão para 2.0.0.

**Architecture:** Três camadas entregues em uma release: (1) tokens — spacing, typography, shadows, radius, motion em `globals.css @theme` + arquivos TypeScript separados; (2) acessibilidade — três falhas Lighthouse corrigidas via `inert`, `role="presentation"` e `aria-label` fix; (3) componentes — renomear props e vincular tokens.

**Tech Stack:** React 19, TypeScript 5, Tailwind CSS v4, Framer Motion 12, tsup, Vitest 4, @testing-library/react 16
